robocopy 指令產生器

把 Windows 內建 robocopy 的旗標變成可勾選面板，即時產出「先預覽（/L）+ 真跑」兩條指令、複製貼到 cmd 跑。內建刪檔防呆。

🔒 純前端、不連外、只產生指令文字（CSP connect-src 'none' 技術強制）。單檔可離線開。
怎麼用：填好來源/目的、勾選需要的選項 → 先複製「預覽指令」貼到 cmd 跑一次（/L 只列出、不動任何檔），看清楚會動到什麼 → 確認沒問題再複製「真跑指令」執行。
路徑可填：本機 C:\資料夾、網路磁碟 \\server\share、USB E:\備份。含空白會自動加引號。

複製範圍

⚠ 刪除 / 移動（會毀資料、預設都不勾）

排除既有檔（控制覆蓋）

⚠ 這三個只影響「複製/覆蓋」、擋不住刪除——同時開 /MIR 或 /PURGE 時，目的端多餘檔照樣被刪。

效能與重試

/R　失敗重試次數 ⚠ 不設預設是 一百萬次、會卡死
/W　重試間隔秒 ⚠ 不設預設是 30 秒

產出指令

① 預覽指令 /L 只列出、不動任何檔（先跑這條看會動到什麼）
② 真跑指令 確認預覽沒問題後再跑這條

robocopy 結束代碼（exit code）怎麼看

⚠ robocopy 跟一般指令相反：0 不是「成功」、是「沒事做」。別用「非 0 即錯」的直覺判讀。

0目的端已經是最新、沒有複製任何檔（正常）
1有檔案成功複製（正常）
2目的端有多餘檔（沒複製動作）
3有複製 + 有多餘檔
≥ 8至少一個檔案複製失敗（這才是真的錯誤）
